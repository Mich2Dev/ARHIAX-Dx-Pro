# ARHIAX DxPro Site

Landing publica de producto para ARHIAX DxPro, propiedad de Sinergia Consulting Group.

## Scripts

```bash
npm install
npm run dev
npm run build
npm run lint
```

## Estructura

- `src/App.tsx`: contenido y secciones de la landing.
- `src/App.css`: direccion visual inspirada en `groupsinergia.com`.
- `public/logo-sinergia.png`: logo oficial de Sinergia usado como marca propietaria.
