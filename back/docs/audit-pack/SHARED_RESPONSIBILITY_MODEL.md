# Shared Responsibility Model

## Packaged by ARHIAX Dx Repo

- governance logic
- policy semantics
- tool catalog
- evidence format
- certificate format
- install blueprint

## Provided by Client at Install Time

- secrets
- provider contracts and quotas
- webhook endpoints
- signing key material
- observability sinks
- deployment platform

## Not Shared

The client should not treat packaged governance controls as editable runtime preferences unless a controlled change process is followed.
